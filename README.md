# 게임 완주 계획표

`index.html`을 브라우저로 열면 실행되는 로컬 웹앱입니다.

## 게임 시간 기준

게임마다 `Main story`, `Main + extra`, `Completionist` 시간을 따로 저장합니다. 일정 계산 기본값은 `Main + extra`이고, 게임 목록의 `M`, `ME`, `C` 버튼으로 게임별 계산 기준을 바꿀 수 있습니다.

## 텍스트 가져오기

아래처럼 한 줄에 게임명, 플랫폼, Main story, Main + extra, Completionist 시간을 탭으로 구분한 형식을 읽습니다.

```text
Limbo	Steam	3시간 34분	4시간 10분	6시간 44분
Enter the Gungeon	Epic	22시간 27분	61시간 51분	160시간 2분
```

플랫폼은 미리 정해진 목록이 아니라 두 번째 칸에 적힌 내용을 그대로 저장합니다. 게임 목록에서는 플랫폼이 `Steam`이면 따로 표시하지 않습니다.

## 플레이 시간 규칙

규칙은 위에서 아래 순서로 적용됩니다. 같은 날짜에 여러 규칙이 걸리면 나중에 추가한 규칙이 우선합니다.

예를 들어 기본은 매일 1시간이고, 2027년부터 2029년까지는 매주 1시간만 가능하다면:

```text
2026-06-30 - 계속: 매일 1시간
2027-01-01 - 2029-12-31: 매주 1시간
```

## 플레이 불가 기간

플레이 불가 기간은 플레이 시간 규칙보다 우선합니다. 예를 들어 매일 1시간 규칙이 있어도 `2027-08-01 - 2027-08-20`을 불가 기간으로 넣으면 그 기간은 0시간으로 계산됩니다.

## 저장과 백업

데이터는 브라우저 저장소에 자동 저장됩니다. 다른 컴퓨터로 옮기거나 백업하려면 앱 안의 JSON 내보내기를 사용하세요. CSV 내보내기는 게임 목록만 저장합니다.
